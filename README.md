Alfred workflow to easily open a iOS Simulator's Safari inspector.

- [Installation](#installation)
- [Credits](#credits)

## Installation

```bash
npm install --global alfred-open-safari-inspector
```

> You may need to set "Allow automation" in Safari developer menu.
